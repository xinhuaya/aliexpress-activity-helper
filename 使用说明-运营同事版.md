# 速卖通活动助手使用说明

统一安装地址：

<https://xinhuaya.github.io/aliexpress-activity-helper/>

> 重要：Tampermonkey 显示“已启用”还不够。必须先在 Chrome 的 Tampermonkey 扩展详情中打开“允许用户脚本”，否则助手不会运行。

## 安装前必须开启权限

1. 在 Chrome 右上角右键点击 Tampermonkey 图标。
2. 点击“管理扩展程序”。
3. 找到“允许用户脚本”，打开这个开关。
4. 如果没有看到该开关，进入 `chrome://extensions`，打开右上角“开发者模式”。
5. 如果 Tampermonkey 编辑器顶部仍显示“请启用‘允许用户脚本’扩展设置”，说明权限尚未开启。

## 安装

1. Chrome 安装 Tampermonkey。
2. 打开上面的统一安装地址。
3. 点击“安装或更新稳定版”。
4. 在 Tampermonkey 页面点击“安装”或“更新”。
5. 回到已经打开的速卖通后台，按 `Ctrl + R` 刷新页面。安装脚本后不刷新，助手不会出现。
6. 刷新完成后，右下角会出现“AE 活动助手”。

脚本 `0.8.6` 支持所有 `aliexpress.com` 子域名，会自动使用当前账号自己的渠道编号，也不要求手动进入商品管理页或营销活动页。

这是最后一次需要统一重新安装。以后负责人发布更高版本时，Tampermonkey 会检查固定更新地址；可以在扩展设置中决定是否自动安装，也可以从 Tampermonkey 菜单手动检查用户脚本更新。

## 一键查询并退出活动

1. 打开任意速卖通卖家后台页面，不需要进入“营销 > 平台活动”。
2. 在右下角“AE 活动助手”输入商品 ID。
3. 保持“预演，不提交”勾选，点击“查报名活动”。
4. 脚本会直接按 ID 读取该商品 `SALE` 标签背后的活动内容，通常只需数秒到几十秒。
5. 检查脚本列出的活动是否正确。
6. 确认无误后，取消勾选“预演，不提交”。
7. 点击“普通退出”。
8. 浏览器会再次弹出确认框，确认后才会提交。
9. 提交后脚本会自动逐个活动跳转，进入“已报名”，搜索商品，选择“库存不足”，点击普通“退出活动”。
10. 如果进入活动页时出现“营销智投计划”介绍弹窗，脚本会自动点“我知道了”并继续。

活动页有两种结构：普通页面使用“商品报名”标签，统一报名入口使用“同意并下一步”。脚本会自动识别并进入“外围活动报名”或“入围活动报名”，无需手动切换步骤。

如果平台改版后脚本仍找不到商品 ID 搜索框，面板会明确提示手动进入正确活动页面：店铺外围进入第 2 步“外围活动报名”，平台入围进入第 4 步“入围活动报名”，然后打开“商品报名 > 已报名”再重试。

脚本只读取这个商品 `SALE` 标签中列出的平台活动和店铺活动，内容与鼠标悬浮时看到的列表相同，不再遍历全店活动。如果没有 `SALE` 标志，就不会生成退出计划。

同一个商品可能同时显示“平台活动/入围”和“店铺活动/外围”。这两类都是独立报名，脚本会分别生成退出项并逐个退出，不会再把平台活动合并掉。

如果平台目录里有多个名称和时间都很接近的活动编号，脚本只会额外核对这些候选中该商品的实际报名记录；仍不能确认时会停止，不会猜测后退出。

活动名称与时间无法唯一匹配时，脚本会停止，避免错退。如果出现速卖通安全验证，脚本也会立即停止，不会绕过验证。

脚本默认跳过已结束活动，只退出未结束的报名活动。退出原因固定为“库存不足”。

## 注意

- 脚本走普通“退出活动”，不会把商品设为“不参加活动商品”。
- 脚本只会自动关闭包含“营销智投计划”的介绍弹窗，不会关闭其他退出确认弹窗。
- 页面返回失败时，以速卖通后台提示为准，脚本不会强退平台不允许退出的活动。
- 不要把账号密码、Cookie、密钥发给别人；脚本只在当前登录的浏览器里运行。

## 安装后没有出现怎么办

1. 检查 Tampermonkey 扩展详情中的“允许用户脚本”是否已经打开。
2. 再按 `Ctrl + R` 刷新当前速卖通后台页面。
3. 点浏览器右上角 Tampermonkey 图标，确认顶部显示“已启用”。
4. 确认脚本名称是 `AliExpress Activity Helper`，版本至少是 `0.8.6`，并且脚本开关已打开。
5. 如果版本低于 `0.8.6`，重新打开统一安装地址并覆盖更新。
6. 仍未出现时，把浏览器地址栏中的完整网址和 Tampermonkey 弹窗截图发给负责人排查；不要发送账号密码。
